Final Project – Data Visualization in Python  
Topic: Portuguese Primeira Liga Statistics 2023/24

This project aims to analyse and visualise data related to players and teams from the Portuguese Primeira Liga (2023/24 season), exploring statistics such as goals, assists, cards, and overall performance throughout the season.

The main focus of the project is exploratory data analysis and the creation of clear and informative visualisations, using both static and interactive charts, as well as the interpretation of the results obtained.

Data source:
Dataset provided on Kaggle – Portugal Primeira Liga 23/24, containing detailed statistics for players and teams.

Main variables analysed:
- Season
- Team
- Player
- Position
- Goals
- Assists
- Yellow and red cards
- Matches played

Analyses performed:
- Descriptive analysis of player and team statistics
- Performance comparison between players and clubs
- Exploration of statistical trends throughout the season
- Creation of static and interactive visualisations
- Interpretation of the results and insights

Technologies used:
- Python
- Pandas
- Matplotlib / Seaborn
- Plotly
- Streamlit

Detailed documentation of the analysis process, methodological decisions, and result interpretation can be found in the project notebook.
